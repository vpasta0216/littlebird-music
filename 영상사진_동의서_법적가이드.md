# 영상/사진 동의서 법적 가이드 (원장용)

> 이 문서는 **원장님이 읽고 이해하기 위한 가이드**입니다.
> 학부모에게 주는 양식은 별도 파일(`영상사진_사용동의서.md`)을 사용하세요.

---

## 1. 왜 동의서가 필요한가?

피아노학원 학생 대부분이 **초등학생(만 14세 미만)**이기 때문에, 영상/사진을 촬영하고 온라인에 올리려면 **법적으로 부모(법정대리인)의 서면 동의**가 필요합니다.

관련 법률:
- **개인정보보호법 제22조의2** — 만 14세 미만 아동의 개인정보 수집 시 법정대리인 동의 필수
- **개인정보보호법 제15조** — 개인정보 수집·이용 시 목적, 항목, 보유기간을 고지하고 동의받아야 함
- **민법 (인격권/초상권)** — 타인의 초상을 허락 없이 촬영·공표하면 초상권 침해

> 동의서 없이 학생 영상을 올리면 **3년 이하 징역 또는 3천만원 이하 벌금**에 해당할 수 있습니다.

---

## 2. 가장 많이 궁금한 것들

### Q1. 퇴원하면 영상을 내려야 하나요?

**아닙니다.** 동의서에 아래 두 가지만 명시하면 퇴원 후에도 계속 사용 가능합니다:

1. **보유기간**을 구체적으로 명시 (예: "동의일로부터 5년")
2. **"수강 종료(퇴원) 후에도 본 동의의 효력은 유지됩니다"** 문구 포함

> 단, 학부모가 **별도로 철회를 요청하면 삭제해야** 합니다.
> 이것은 법적으로 막을 수 없는 권리입니다 (개인정보보호법 제36조).
> 하지만 실제로 퇴원 후 철회를 요청하는 경우는 거의 없습니다.

### Q2. "영구 사용"으로 쓸 수 있나요?

**"영구"라는 표현은 피하세요.** 개인정보보호법상 보유기간은 구체적으로 명시해야 합니다.

대신 이렇게 쓰면 됩니다:

| 표현 | 적합 여부 |
|------|-----------|
| "영구" | X — 법적으로 인정 어려움 |
| "동의 철회 시까지" | △ — 기간이 특정되지 않아 논란 여지 |
| **"동의일로부터 5년"** | **O — 가장 안전하고 실무적** |
| "동의일로부터 10년" | O — 가능하지만 5년이 일반적 |

> **실무 팁:** 5년으로 설정하고, 5년 후에 재등록 학생이면 자연스럽게 재동의를 받으면 됩니다. 퇴원한 학생의 경우 5년이면 대부분 충분합니다.

### Q3. 플랫폼별로 따로 동의받아야 하나요?

**아닙니다.** 사용 범위를 포괄적으로 쓰면 됩니다:

```
학원이 운영하는 온·오프라인 매체 전체
(홈페이지, SNS, 유튜브, 블로그, 인쇄물 및 향후 개설 채널 포함)
```

이렇게 쓰면 홈페이지, 인스타, 유튜브, 네이버 블로그, 카카오톡 채널, 나중에 새로 만드는 채널까지 **한 번의 동의로 전부 커버**됩니다.

> **"향후 개설 채널 포함"** 이 문구가 핵심입니다.

### Q4. 얼굴/손 따로 나눠야 하나요?

**나눌 필요 없습니다.** 동의하는 학부모는 전체 동의로 받고, 동의하지 않는 학부모는 아예 서명을 안 하면 됩니다.

동의서에는 이렇게 한 줄이면 충분합니다:

```
수업, 발표회, 학원 행사 중 촬영된 사진 및 영상 일체
```

> 실무적으로 얼굴/손/뒷모습을 나누면 관리가 너무 복잡해집니다.
> "동의 O / 비동의 X" 두 가지로만 운영하는 것이 현실적입니다.

## 3. 동의서 작성 시 법적 필수 요소

개인정보보호법에 따라 동의서에 **반드시 들어가야 하는 항목**:

| 필수 항목 | 내용 | 법적 근거 |
|-----------|------|-----------|
| **수집 목적** | 학원 홍보 목적의 사진/영상 촬영 및 게시 | 제15조 제1항 |
| **수집 항목** | 사진, 영상, 성(이니셜) | 제15조 제1항 |
| **이용 범위** | 온·오프라인 매체 전체 | 제15조 제1항 |
| **보유 기간** | 동의일로부터 5년 | 제15조 제1항 |
| **동의 거부권** | 동의하지 않을 수 있음을 안내 | 제16조 제2항 |
| **철회 방법** | 구두/문자/카톡 등으로 언제든 철회 가능 | 제36조 |

## 4. 반드시 지켜야 할 주의사항

### 절대 안 되는 것

| 행위 | 위반 법률 | 벌칙 |
|------|-----------|------|
| 동의 없이 영상 게시 | 개인정보보호법 제15조 | 5천만원 이하 과태료 |
| 아동 동의만 받고 부모 동의 안 받음 | 개인정보보호법 제22조의2 | 5천만원 이하 과태료 |
| 학원 홍보 외 목적으로 사용 | 개인정보보호법 제18조 | 5년 이하 징역, 5천만원 이하 벌금 |
| 학부모 철회 요청 무시 | 개인정보보호법 제36조 | 5천만원 이하 과태료 |

### 꼭 해야 하는 것

- 미동의 학생은 **촬영 자체를 피하거나**, 불가피하면 **편집으로 제거**
- 학생 이름은 **성+이니셜**(김○○)만 사용, 풀네임 공개 금지
- 학교명, 학년 등 **특정 가능한 정보 최소화**
- 동의서 원본은 **5년간 보관** (분쟁 시 증거)
- 철회 요청 시 **7일 이내 삭제 후 통보**

## 5. 실무 운영 팁

### 동의서 배포 시기

| 시점 | 방법 |
|------|------|
| **신규 등록 시** | 등록 서류와 함께 동의서 배포 → 서명 |
| **기존 학생** | 카톡/대면으로 안내 후 배포 → 1~2주 내 수거 |
| **5년 경과 시** | 재등록 학생에게 재동의 요청 |

### 동의 현황 관리 (엑셀/구글시트)

| 학생명 | 등록일 | 동의여부 | 동의일 | 만료일 | 비고 |
|--------|--------|----------|--------|--------|------|
| 김○○ | 2026.03 | O | 2026.03.05 | 2031.03.05 | |
| 이○○ | 2026.03 | X | - | - | 촬영 불가 |
| 박○○ | 2026.04 | O | 2026.04.10 | 2031.04.10 | 2027.06 퇴원, 영상 유지 |

### 촬영 시 체크

1. 촬영 전 미동의 학생 확인
2. 미동의 학생이 같은 시간대면 → 카메라 각도 조절 또는 해당 시간 촬영 안 함
3. 단체 촬영 시 미동의 학생은 프레임 밖에 배치
4. 편집 시 미동의 학생이 나왔으면 해당 부분 컷 또는 블러 처리
