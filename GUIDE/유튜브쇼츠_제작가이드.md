# 유튜브 쇼츠(Shorts) 제작 가이드 (초보용)

> 쇼츠는 **60초 이하의 세로 짧은 영상**입니다. 인스타 릴스와 비슷해요!
> 짧고 임팩트 있어서 조회수가 잘 나옵니다. 피아노학원에 딱 맞는 콘텐츠예요!

---

## 1. 쇼츠란?

- **길이:** 60초 이하 (15~30초가 가장 좋음)
- **화면:** 세로 영상 (핸드폰 세워서 촬영)
- **특징:** 유튜브 앱에서 틱톡처럼 위아래로 넘기면서 봄
- **장점:** 구독자가 없어도 수천~수만 조회수 가능!

### 일반 영상 vs 쇼츠 비교

| | 일반 영상 | 쇼츠 |
|---|-----------|-------|
| **길이** | 제한 없음 | 60초 이하 |
| **화면** | 가로 | 세로 |
| **조회수** | 천천히 오름 | 빠르게 터질 수 있음 |
| **용도** | 자세한 연주 영상 | 짧고 임팩트 있는 영상 |
| **촬영** | 핸드폰 옆으로 | 핸드폰 세워서 |

## 2. 피아노학원 쇼츠 아이디어 20가지

### 연주 영상 계열 (가장 쉽고 반응 좋음!)

1. **학생 연주 하이라이트** - 가장 멋진 부분 15~30초
2. **Before & After** - 처음 vs 3개월 후 (학부모들이 정말 좋아함!)
3. **선생님 연주** - 누나가 직접 연주 (실력 어필 + 신뢰도)
4. **선생님 vs 학생 듀엣** - 포핸즈로 같이 치는 모습
5. **한 곡의 클라이맥스** - 곡의 가장 멋진 부분만 15초

### 일상/비하인드 계열 (친근함 어필)

6. **학원 하루 일과 타임랩스** - 빈 레슨실 → 학생들 오고 → 마무리
7. **수업 준비하는 모습** - 피아노 닦기, 악보 준비
8. **학생이 처음 "완곡"했을 때 반응** - 기뻐하는 모습
9. **학원 간식타임** - 쉬는 시간에 간식 먹는 모습
10. **학원 인테리어/분위기 소개** - 느린 카메라로 구석구석

### 교육/정보 계열 (전문성 어필)

11. **"이 곡 아시나요?" 퀴즈** - 3초 연주 후 정답 공개
12. **피아노 자세 교정 팁** - "이렇게 치면 안 돼요 ❌ → 이렇게 ⭕"
13. **음표 읽는 법** - "도레미파솔을 30초만에!"
14. **연습 방법 팁** - "이 부분이 안 될 때는 이렇게"
15. **피아노 관리 팁** - "피아노 이렇게 관리하세요"

### 감성/스토리 계열 (공감 유발)

16. **"피아노 선생님의 하루"** - 1인칭 시점
17. **"1년 전 입학생의 성장기"** - 사진/영상 모아서 성장 과정
18. **발표회 비하인드** - 무대 뒤 떨리는 모습 → 무대에서 멋지게!
19. **"이 곡을 연습한 시간: 3개월"** - 연습 과정을 빠르게
20. **졸업하는 학생에게 보내는 메시지** - 감동적인 내용

## 3. 쇼츠 촬영 방법

### 촬영 기본 규칙

| 규칙 | 설명 |
|------|------|
| **세로로 촬영!** | 핸드폰 세워서. 가로로 찍으면 쇼츠로 안 됩니다 |
| **15~30초가 최적** | 짧을수록 끝까지 보는 비율이 높아짐 |
| **첫 1~2초가 승부** | 시작부터 연주 소리 or 눈길 끄는 장면. "안녕하세요~" 인사로 시작하면 넘겨버림 |
| **지루한 구간 없이** | 잘 된 부분만 잘라서 편집 |
| **자막 넣기** | 소리 없이 보는 사람도 많으니까 |

### 구도 추천

| 유형 | 구도 |
|------|------|
| **연주 영상** | 핸드폰을 세워서 피아노 앞 45도 각도. 건반 + 손 + 상반신이 보이게. 또는 손/건반만 클로즈업 |
| **학원 소개** | 천천히 걸으면서 촬영 또는 한 곳에 세워놓고 타임랩스 |
| **팁/교육** | 정면에 세워놓고 말하기 또는 건반을 위에서 내려다보며 촬영 |

## 4. 쇼츠 편집 방법 (무료 앱)

### 추천 앱 1: CapCut (캡컷) - 가장 추천!

- **무료**, 사용법이 쉬움, 자막 자동 생성 기능 있음!
- 다운로드: 앱스토어에서 **"CapCut"** 검색

**기본 편집 방법:**

1. CapCut 앱 설치하고 열기
2. **"새 프로젝트"** 클릭
3. 갤러리에서 영상 선택
4. 아래 편집 도구 사용:

| 기능 | 설명 |
|------|------|
| **자르기** | 영상의 시작점/끝점을 잘라서 15~30초로 |
| **자막** | "텍스트" > "자동 자막" 누르면 자동 생성! |
| **음악** | 배경음악 추가 (필요한 경우만) |
| **필터** | 밝기/색감 조절 |
| **속도** | 타임랩스 만들 때 사용 (2배속, 4배속 등) |

5. 오른쪽 위 **내보내기(↑)** 클릭 → 1080p로 저장

### 추천 앱 2: 유튜브 앱 자체 편집

1. 유튜브 앱 > [+] > **"쇼츠 만들기"**
2. 갤러리에서 영상 선택 (또는 바로 촬영)
3. 자르기, 음악, 텍스트 추가 가능
4. 바로 업로드!

> 간단한 연주 영상은 이 방법이 가장 빠름!

### 추천 앱 3: VLLO (블로)

- **한국어 앱**이라 메뉴가 편함
- 무료 (일부 기능 유료)
- 자막 넣기 편함

## 5. 쇼츠 업로드 방법

1. 유튜브 앱 열기
2. 아래쪽 가운데 **[+]** 버튼 클릭
3. **"쇼츠 만들기"** 선택
4. 왼쪽 아래 갤러리 아이콘 → 편집된 영상 선택
5. 필요하면 추가 편집 (텍스트, 음악)
6. **"다음"** 클릭
7. 정보 입력:

### 제목 쓰는 법

| 구분 | 예시 |
|------|------|
| **좋은 예** | "초등2학년이 3개월 만에 이 곡을?! 🎹" |
| **좋은 예** | "피아노 1년 배우면 이 정도 칩니다 ✨" |
| **좋은 예** | "선생님도 감동한 학생의 첫 발표회 🥹" |
| **좋은 예** | "이 자세로 치면 안 돼요! ❌ #피아노팁" |
| **좋은 예** | "Before & After \| 피아노 6개월의 변화 🎵" |
| **나쁜 예** | "연주 영상", "학생 피아노", "2026-02-18 수업" |

**해시태그 (제목 아래에 붙여주세요):**

```
#피아노 #피아노연주 #울산피아노학원
#피아노레슨 #초등학생피아노 #피아노shorts
#piano #pianocover #pianostudent
```

8. 공개 설정 → "공개" 선택
9. **"쇼츠 업로드"** 클릭하면 끝!

## 6. 쇼츠 + 인스타 릴스 동시에 올리기

쇼츠와 인스타 릴스는 같은 세로 영상이라 **하나 만들면 두 곳에 다 올릴 수 있어요!**

### 1개 만들고 2곳에 올리기 루틴

1. 영상을 **세로로 촬영**한다
2. **CapCut으로 편집**한다 (자막, 자르기 등)
3. 편집된 영상을 **저장**한다
4. **유튜브 쇼츠**로 업로드 (제목 + 해시태그)
5. 같은 영상을 **인스타그램 릴스**로도 업로드 (인스타 앱 > [+] > 릴스 > 영상 선택 > 공유)

> **한 번의 촬영 + 편집으로 2곳에 콘텐츠 완성!**

### 주의사항

- 유튜브 쇼츠에서 음악을 추가했다면, 인스타에 올릴 때는 인스타 자체 음악으로 다시 넣어야 합니다 (저작권이 다름)
- **가장 안전한 방법:** 피아노 연주 영상은 자체 소리가 음악이니까 별도 배경음악 없이 올리면 양쪽 다 문제 없음!

## 7. 쇼츠 잘 만드는 핵심 팁

### 조회수 올리는 5가지 법칙

| 법칙 | 설명 |
|------|------|
| **1. 첫 1초가 전부** | 시작부터 연주 소리 or 자막 or 궁금증 유발. "안녕하세요~" ❌ |
| **2. 15~20초가 최적** | 짧을수록 완시율이 높아지고, 유튜브가 더 많이 추천 |
| **3. 반복 재생되게** | 끝이 처음과 자연스럽게 이어지면 반복 재생 = 조회수 증가 |
| **4. 자막을 꼭 넣기** | 소리 없이 보는 사람이 60% 이상. CapCut "자동 자막" 활용 |
| **5. 해시태그 활용** | 5~10개 정도가 적당 |

### 피해야 할 것들

- ❌ 너무 긴 인트로 (로고 애니메이션 등)
- ❌ "좋아요 구독 눌러주세요" (쇼츠에서는 효과 없음)
- ❌ 영상 중간에 지루한 구간
- ❌ 화질 나쁜 영상
- ❌ 소리가 안 들리는 영상

## 8. 실전 예시: 쇼츠 하나 만들어보기

> 연습 삼아 **"학생 연주 하이라이트"** 쇼츠를 만들어봅시다!

**준비물:** 학생 연주 영상 1개 (핸드폰에 있는 것) + CapCut 앱

### 따라하기

1. CapCut 앱을 열고 **"새 프로젝트"** 클릭
2. 학생 연주 영상을 선택
3. 영상에서 **가장 잘 된 부분 15~25초**를 찾기
   - 타임라인을 좌우로 밀면서 확인
   - 시작점에서 "분할" → 끝점에서 "분할"
   - 필요 없는 앞뒤 부분 선택 후 "삭제"
4. **텍스트(자막) 추가**
   - 아래 메뉴에서 "텍스트" → "텍스트 추가"
   - 입력: `초등○학년 | 피아노 ○개월차`
   - 위치를 화면 상단이나 하단에 배치
5. (선택) 마지막에 학원 이름 자막 추가
   - `🎹 ○○피아노학원 | 울산` → 마지막 3초에만
6. **내보내기(↑)** 클릭 → 해상도 1080p → 저장
7. **유튜브 쇼츠로 업로드** → 제목: "초등○학년이 ○개월 만에 이렇게! 🎹 #피아노"
8. **같은 영상을 인스타 릴스로도 업로드**

> 첫 쇼츠 완성!
