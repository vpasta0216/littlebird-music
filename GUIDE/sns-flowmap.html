1 채널 역할 정의

🌐

홈페이지

"학원의 얼굴"

커리큘럼, 강사 소개, 위치 안내. 검색(네이버/구글)으로 처음 찾아오는 사람이 가장 먼저 보는 곳.

📸

인스타그램

"일상 공유 창구"

수업 사진/영상, 릴스, 이벤트 포스터. 비팔로워에게도 노출되어 새 학부모 유입에 효과적.

🎬

유튜브

"영상 저장소"

발표회 풀영상, 학생 연주, 쇼츠. 긴 영상 보관 + 짧은 쇼츠로 노출 확대.

💬

카카오채널

"학부모 알림"

공지 알림, 이벤트 안내, 1:1 상담. 자동응답으로 수업 중에도 문의 대응.

2 콘텐츠 흐름 매핑 지도

흐름 1 영상 콘텐츠

영상 촬영 편집 (CapCut)
├ 유튜브 (긴 영상)
├ 인스타 릴스 (짧은 클립)
└ 유튜브 쇼츠 (세로 짧은 영상)
핵심: 세로로 촬영하면 릴스 + 쇼츠에 동시 활용 가능! 한 번 만들면 두 곳에 올려요.

흐름 2 사진 / 포스터

사진/포스터 제작 (Canva)
├ 인스타 피드 게시
├ 카카오채널 알림 발송
└ 홈페이지 업데이트

흐름 3 이벤트 (신학기 / 방학 / 발표회)

이벤트 기획
├ 포스터 제작
├ 인스타 게시
├ 카카오 알림
└ 홈페이지 배너
└ 영상 촬영 (발표회)
├ 유튜브 풀영상
├ 인스타 릴스 (하이라이트)
└ 유튜브 쇼츠

3 시나리오별 체크리스트

시나리오 1 일상 클립

수업 풍경, 연습 영상 등 일상적인 짧은 콘텐츠
  1. 1 핸드폰 세로로 촬영 (15~30초 하이라이트)
  2. 2 CapCut에서 편집 (자르기 + 자막 추가)
  3. 3 인스타 릴스로 업로드 + 유튜브 쇼츠로 업로드
  4. 4 카카오 "오늘의 수업 영상 올렸어요!" 알림 발송

시나리오 2 이벤트

신학기 모집, 방학 특강, 할인 이벤트 등
  1. 1 Canva에서 이벤트 포스터 제작
  2. 2 인스타 피드 게시 (해시태그 세트 복사 붙여넣기)
  3. 3 카카오 소식 발송 (이미지 + 간략 안내)
  4. 4 홈페이지 배너 교체 / 공지 업데이트
  5. 5 이벤트 종료 후 결과 사진 인스타에 업로드

시나리오 3 발표회

정기 발표회, 연주회 등 대형 행사
  1. 1 발표회 전체 영상 촬영 (가로) + 하이라이트 촬영 (세로)
  2. 2 편집: 풀영상 + 학생별 개인 클립 + 하이라이트 릴스/쇼츠
  3. 3 유튜브 풀영상 업로드 (재생목록 "발표회"에 추가)
  4. 4 인스타 릴스 하이라이트 + 쇼츠 업로드
  5. 5 카카오 학부모에게 "영상 올렸어요!" 알림 + 유튜브 링크 공유
  6. 6 홈페이지 갤러리에 베스트 영상 추가

4 실전 방법 요약 (3단계로 끝!)

인스타 릴스 올리기

  1. 인스타 앱 열고 하단 [+] 버튼 누르기 > "릴스" 선택
  2. 갤러리에서 영상 선택 (15~90초) > 음악/텍스트 추가
  3. 캡션 + 해시태그 붙여넣기 > "공유" 누르면 끝!

유튜브 영상 올리기

  1. YouTube 앱 열고 하단 [+] 버튼 > "동영상 업로드"
  2. 제목 + 설명 입력 (템플릿 복사) > 썸네일 선택
  3. 공개 설정 > "업로드" 누르면 끝! (Wi-Fi 권장)

유튜브 쇼츠 만들기

  1. YouTube 앱 열고 [+] 버튼 > "쇼츠 만들기"
  2. 갤러리에서 세로 영상 선택 (60초 이내) > 텍스트/음악 추가
  3. 제목 + #Shorts 해시태그 > "쇼츠 업로드"로 완료!

카카오채널 알림 보내기

  1. 카카오톡 채널 관리자 앱 > "소식" > "새 소식"
  2. 이미지 + 제목 + 내용 작성
  3. 발송 대상 선택 > "발송" 누르면 끝!
