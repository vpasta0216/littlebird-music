# 유튜브 채널 만들기 & 운영 가이드 (완전 초보용)

> 유튜브를 한 번도 안 해본 분도 따라할 수 있도록 만든 가이드입니다.
> 천천히 순서대로 따라하시면 됩니다!

---

## STEP 1. 구글 계정 만들기

> 이미 Gmail을 쓰고 있다면 구글 계정이 있는 거예요! → STEP 2로 건너뛰세요.

### 구글 계정이 없는 경우

1. 브라우저에서 이동: **https://accounts.google.com/signup**
2. 이름 입력 (학원 이름으로 해도 됩니다)
3. 이메일 주소 만들기 (예: `doremi.piano.ulsan@gmail.com`)
4. 비밀번호 설정
5. 전화번호 인증하면 완성!

> **팁:** 개인 계정과 분리하고 싶으면 학원용 계정을 새로 만드세요.
> 귀찮으면 기존 계정을 써도 됩니다 (채널은 별도로 만들어지니까 개인 정보 노출 안 됨).

## STEP 2. 유튜브 채널 만들기

1. **https://www.youtube.com** 접속
2. 오른쪽 위 동그란 아이콘 클릭 → "로그인" → 구글 계정으로 로그인
3. 다시 오른쪽 위 아이콘 클릭 → **"채널 만들기"** 클릭
4. 채널 이름 입력:

> **추천 채널 이름:** ○○피아노학원 / ○○ Piano Academy / ○○피아노 울산
> 검색이 잘 되려면 **"학원이름 + 피아노"** 조합이 좋아요!

5. 핸들(고유 주소) 설정: 영어로 짧게 (예: `@doremi-piano-ulsan`)
6. **"채널 만들기"** 클릭하면 완성!

## STEP 3. 채널 꾸미기

1. 유튜브 > 오른쪽 위 아이콘 > **"내 채널"** 클릭
2. **"채널 맞춤설정"** 클릭

### 프로필 사진
- 학원 로고 또는 학원 대표 사진
- 정사각형 이미지 (800x800 이상 권장)
- 카카오톡 채널과 같은 사진을 쓰면 통일감!

### 배너 이미지
- 채널 상단에 크게 보이는 가로 이미지
- 크기: **2048 x 1152 픽셀** 권장
- 넣으면 좋은 내용: 학원 이름, 간단한 소개, 전화번호

> **만들기 어려우면?** → **Canva(캔바)** 무료 사이트에서 만들 수 있어요!
> https://www.canva.com 접속 → "유튜브 배너" 검색 → 예쁜 템플릿 골라서 글자만 바꾸면 끝!

## STEP 4. 영상 올리는 방법 (업로드)

### 방법 A: 핸드폰에서 올리기 (가장 쉬움!)

1. 핸드폰에서 **"유튜브"** 앱 열기 (없으면 앱스토어에서 설치)
2. 구글 계정으로 로그인
3. 화면 아래쪽 가운데 **[+]** 버튼 누르기
4. **"동영상 업로드"** 선택
5. 갤러리에서 올릴 영상 선택
6. 아래 정보를 입력:

#### 제목 쓰는 법

| 구분 | 예시 |
|------|------|
| **좋은 예** | "초등2학년 피아노 연주 \| 엘리제를 위하여 \| ○○피아노학원" |
| **좋은 예** | "피아노 발표회 \| 2026년 봄 연주회 \| 울산 ○○피아노" |
| **좋은 예** | "바이엘 100번 연주 \| 6개월 배운 학생 \| ○○피아노학원" |
| **나쁜 예** | "영상1", "ㅎㅎ 잘했어", "2026-02-18" |

> **제목 공식:** `[곡이름 또는 내용] | [학생정보] | [학원이름]`

#### 설명 템플릿 (매번 복사해서 곡 정보만 바꾸기)

```
🎹 ○○피아노학원 학생 연주 영상

곡명: (곡 이름)
연주: (학생 이름 또는 "○학년 학생")
교육기간: (예: 피아노 시작 6개월)

━━━━━━━━━━━━━━
○○피아노학원
📍 울산 ○○구 ○○동 ○○아파트 상가 ○층
📞 052-○○○-○○○○
💬 카카오톡: @○○○
📷 인스타그램: @○○○

#울산피아노학원 #피아노연주 #초등학생피아노
#피아노레슨 #○○피아노학원
```

#### 공개 설정

| 설정 | 설명 |
|------|------|
| **공개** | 누구나 볼 수 있음 (일반 영상은 이걸로) |
| **일부공개** | 링크가 있는 사람만 볼 수 있음 (특정 학부모에게만) |
| **비공개** | 나만 볼 수 있음 |

7. 오른쪽 위 **"업로드"** 버튼 누르면 끝!

> Wi-Fi에서 하는 게 좋습니다 (데이터 많이 씀!)

### 방법 B: 컴퓨터에서 올리기

1. 핸드폰 영상을 컴퓨터로 옮기기 (USB 케이블, 카카오톡 나와의 채팅, 구글 드라이브 등)
2. **https://studio.youtube.com** 접속
3. 오른쪽 위 **"만들기"** → **"동영상 업로드"**
4. 영상 파일을 끌어다 놓거나 "파일 선택" 클릭
5. 제목, 설명 입력 (핸드폰과 동일)
6. **"다음"** → **"다음"** → **"다음"** → 공개 설정 → **"게시"**

## STEP 5. 재생목록 만들기 (영상 정리하기)

### 추천 재생목록 구성

| 재생목록 이름 | 용도 |
|--------------|------|
| 🎹 학생 연주 영상 | 학생들의 개별 연주 영상 모음 |
| 🎵 발표회 / 연주회 | 발표회 영상 모음 (연도별로 나눠도 좋음) |
| 📚 커리큘럼 소개 | 단계별로 어떤 곡을 치는지 보여주는 영상 |
| 🏫 학원 소개 | 학원 내부 투어, 분위기 소개 영상 |
| 🎬 Shorts (쇼츠) | 짧은 영상 모음 |

### 만드는 방법

1. 유튜브 스튜디오 접속 (https://studio.youtube.com)
2. 왼쪽 메뉴 **"재생목록"** 클릭
3. 오른쪽 위 **"새 재생목록"** 클릭
4. 이름 입력 → 공개 설정 "공개" → **"만들기"**

> 영상을 올릴 때 재생목록을 선택할 수 있고, 이미 올린 영상도 나중에 추가 가능!

## STEP 6. 4년치 기존 영상 정리해서 올리기

### 1단계: 영상 분류하기

| 폴더 | 내용 |
|------|------|
| **폴더 A** | 올려도 되는 영상 (학부모 동의 받은 것) |
| **폴더 B** | 동의 확인 필요한 영상 |
| **폴더 C** | 비공개로 올릴 영상 (기록용) |
| **폴더 D** | 안 올릴 영상 (화질 나쁨, 의미 없음 등) |

### 2단계: 올리는 순서

| 시기 | 할 일 |
|------|-------|
| **1주차** | 가장 잘 된 영상 5~10개 먼저 올리기 (첫인상용) |
| **2주차** | 발표회/연주회 영상 올리기 (연도별 재생목록 정리) |
| **3주차~** | 나머지 영상 천천히 올리기 (하루에 2~3개씩) |

> **꿀팁:** 한 번에 50개 올리는 것보다 매일 2~3개씩 올리는 게 유튜브 알고리즘에 유리합니다!

## STEP 7. 홈페이지 갤러리 연동 방식

```
┌──────────────┐
│  유튜브에 영상    │ ← 영상은 여기에 전부 올림
│  업로드 (전체)    │
└──────┬───────┘
       │ 골라서 표시
       ▼
┌──────────────┐
│  홈페이지 갤러리   │ ← 대표 영상 5~8개만 보여줌
│  (엄선된 영상만)   │
│                │
│  [더보기] 버튼  │ ── 클릭하면 → 유튜브 채널로 이동
└──────────────┘
```

### 누나가 할 일

1. 유튜브에 영상을 올린다
2. 갤러리에 넣고 싶은 영상이 있으면 알려준다 ("이 영상 홈페이지에도 올려줘")
3. 제가 홈페이지 갤러리에 추가

> **또는** 유튜브 "홈페이지 갤러리" 재생목록을 만들어서, 거기에 넣은 영상만 자동으로 홈페이지에 보이게 할 수도 있어요!

## STEP 8. 유튜브 → 인스타그램 공유하는 방법

### 방법 1: 직접 공유 (가장 확실함, 추천!)

1. 핸드폰 갤러리에서 **원본 영상을 인스타에도 올리기**
   - 인스타 앱 > [+] > 릴스 또는 게시물 > 영상 선택 > 공유
2. 인스타 게시물 설명에 유튜브 링크 적기:

```
🎹 ○○피아노학원 학생 연주

곡명: 엘리제를 위하여
연주: 초등3학년 (피아노 1년차)

전체 영상은 유튜브에서 보실 수 있어요!
🔗 프로필 링크에서 유튜브 채널 방문

#울산피아노학원 #피아노연주 #초등학생피아노
#피아노레슨 #울산 #○○피아노학원
#피아노 #pianocover #pianostudent
```

### 방법 2: 자동 공유 도구 (IFTTT)

1. https://ifttt.com 접속 → 무료 회원가입
2. "Create" 클릭 → "유튜브에 새 영상 올리면 → 인스타에 자동 게시"

> 설정이 복잡할 수 있으니 필요하시면 도와드리겠습니다!

### 방법 3: 홈페이지에서 자동 처리

홈페이지에 글/영상을 올리면 → 인스타그램에 자동 공유. 홈페이지 개발할 때 설정해드리겠습니다!

> **현실적인 추천:** 처음에는 **방법 1 (직접 공유)**이 가장 쉽고 확실합니다! 익숙해지면 자동화로 전환.

## STEP 9. 인스타그램 비즈니스 계정 전환 (자동 공유용)

> 홈페이지↔인스타 자동 연동을 위해 필요합니다. 전환은 **무료**이고 간단!

1. 인스타그램 앱 열기
2. 오른쪽 아래 프로필 아이콘 클릭
3. 오른쪽 위 햄버거 메뉴(≡) 클릭
4. **"설정 및 활동"** 클릭
5. **"계정 유형 및 도구"** 클릭
6. **"비즈니스 계정으로 전환"** 클릭
7. 카테고리 선택: "교육" 또는 "음악 레슨/학원"
8. 연락처 정보 입력 (전화번호, 이메일, 주소)
9. **"완료"** 클릭하면 끝!

**비즈니스 계정의 장점:**
- 홈페이지에서 인스타 자동 공유 가능
- "전화하기" 버튼이 프로필에 표시됨
- 게시물 통계를 볼 수 있음 (몇 명이 봤는지)
- 무료!

## STEP 10. 운영 루틴

### 주 1회 루틴 (약 30분)

1. 이번 주에 촬영한 영상 중 좋은 것 1~2개 골라서
2. 유튜브에 업로드 (제목, 설명 양식대로)
3. 같은 영상을 인스타그램에도 올리기
4. 끝!

### 월 1회 루틴 (약 1시간)

1. 이번 달 올린 영상 중 베스트 1~2개를 홈페이지 갤러리에 추가
2. 재생목록 정리 (새 영상 분류)
3. 채널 설명이나 프로필 업데이트할 게 있으면 수정

### 분기 1회 (3개월마다)

1. 발표회 영상이 있으면 편집해서 업로드
2. 채널 전체적으로 정리
3. 홈페이지 갤러리 대표 영상 교체

## STEP 11. 영상 촬영 팁

> 비싼 장비 없어도 됩니다! **핸드폰이면 충분해요.**

### 촬영 기본 규칙

| 규칙 | 설명 |
|------|------|
| **가로로 촬영** | 유튜브 일반 영상은 반드시 가로 (핸드폰 옆으로 눕혀서). 세로 영상은 쇼츠/릴스용 |
| **밝은 곳에서** | 형광등이라도 밝으면 OK. 역광(창문 앞) 피하기 |
| **소리가 잘 들리게** | 핸드폰을 피아노에서 1~2미터 거리에 놓기 |
| **흔들리지 않게** | 핸드폰 거치대 사용 추천 (다이소 5,000원) |
| **구도** | 학생의 손과 건반이 잘 보이게, 옆에서 45도 각도가 가장 예쁨 |

### 추천 촬영 장비 (저렴한 것들)

| 장비 | 가격 | 필수 여부 |
|------|------|-----------|
| 핸드폰 거치대 (삼각대형) | 5,000~15,000원 | **필수** |
| 링라이트 (LED 조명) | 15,000~30,000원 | 있으면 좋음 |
| 핸드폰 마이크 | 10,000~30,000원 | 있으면 좋음 |
| 카메라 (미러리스) | 50만원~ | 나중에 여유되면 |
