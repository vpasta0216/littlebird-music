### 1. 개인정보 및 초상권 (가장 중요!)

#### 반드시 지켜야 할 것

| 항목 | 내용 |
|------|------|
| **동의서 필수** | 학생 영상/사진을 온라인에 올리려면 반드시 **학부모 서면 동의** 필요 |
| **법적 근거** | 개인정보보호법 제22조 - 만 14세 미만 아동은 법정대리인 동의 필수 |
| **초상권** | 타인의 얼굴이 나오는 사진/영상은 동의 없이 게시하면 초상권 침해 |
| **동의 철회** | 학부모가 언제든 동의를 철회할 수 있어야 함 |

#### 실무 체크리스트

- [ ] 모든 학생에 대해 영상/사진 동의서 받기
- [ ] 동의서 원본은 학원에 보관 (분실 주의)
- [ ] 미동의 학생은 별도 표시 → 촬영 시 프레임 밖으로
- [ ] 퇴원 학생의 영상은 동의 범위 재확인
- [ ] 동의 철회 요청 시 7일 이내 해당 콘텐츠 삭제

#### 꼭 하지 말아야 할 것

- 동의 없이 학생 얼굴이 나오는 영상 게시
- 학생의 이름(풀네임) 공개
- 학생의 학교, 학년 등 특정 가능한 정보 과도하게 노출
- "이 학생은 ○○초등학교 3학년" 같은 표현

#### 안전한 표현 방법

| 위험한 표현 | 안전한 표현 |
|-------------|-------------|
| 김민수 (○○초 3학년) | 초등 3학년 민○ |
| ○○초등학교 학생 | 초등학생 |
| 김민수 학생의 연주 | ○○이의 연주 (동의받은 경우) |
| 전체 이름 노출 | 성+이니셜 또는 별명 |

### 2. 음악 저작권

#### 피아노 연주 영상의 저작권

| 상황 | 저작권 여부 | 대응 |
|------|------------|------|
| **클래식 곡 연주** (베토벤, 쇼팽 등) | 저작권 만료 (사후 70년) → **자유롭게 사용 가능** | 문제 없음 |
| **최신 가요/팝송 커버** | 저작권 있음 → 주의 필요 | 아래 참조 |
| **교재 수록곡** (체르니, 하농 등) | 대부분 저작권 만료 → **자유롭게 사용 가능** | 문제 없음 |
| **영화/드라마 OST** | 저작권 있음 → 주의 필요 | 아래 참조 |
| **동요** | 대부분 저작권 만료 → 확인 필요 | 곡별 확인 |

#### 유튜브에서의 저작권 처리

- **유튜브**: 저작권이 있는 곡을 연주하면 자동 감지됨 → 보통 광고가 붙거나 수익이 원저작자에게 감 → 영상 삭제는 드묾
- **인스타그램**: 유튜브보다 관대하지만 음원 자체를 사용하면 삭제될 수 있음
- **학생이 직접 연주한 것**: 실연(演)의 권리는 연주자에게 → 학부모 동의만 있으면 대부분 문제 없음

#### 안전한 방법

1. **클래식 곡 위주로 업로드** (가장 안전)
2. 최신곡 커버는 **유튜브에만** 올리기 (저작권 시스템이 자동 처리)
3. 인스타에 최신곡 올릴 때는 **인스타 자체 음원 라이브러리** 사용
4. 저작권 걱정되면 **"피아노 연주"라고 명시** + 원곡 크레딧 표기

### 1. 단계별 런칭 전략

> 한꺼번에 다 하면 지칩니다! 아래 순서대로 하나씩 진행하세요.

#### 1단계: 기반 준비 (1~2주)

- [ ] 학원 정보 작성 가이드 완성
- [ ] 영상/사진 동의서 배포 및 수거
- [ ] 카카오톡 채널 개설 + 자동응답 설정
- [ ] 인스타그램 비즈니스 계정 생성 + 프로필 세팅

#### 2단계: 콘텐츠 준비 (1~2주)

- [ ] 기존 영상 중 쇼츠/릴스용 3~5개 편집
- [ ] 학원 사진 촬영 (시설, 수업 모습)
- [ ] Canva로 학원 소개 카드뉴스 1개 제작
- [ ] 인스타그램 게시물 3~5개 미리 준비

#### 3단계: SNS 오픈 (1주)

- [ ] 인스타그램 첫 게시물 업로드 (학원 소개)
- [ ] 학생 연주 릴스 1~2개 업로드
- [ ] 학부모 안내문 발송 (카카오톡)
- [ ] 네이버 블로그 개설 + 첫 글 발행
- [ ] 유튜브 채널 개설 + 기존 영상 업로드 시작

#### 4단계: 홈페이지 오픈 (2~4주)

- [ ] 학원 정보 기반으로 홈페이지 내용 완성
- [ ] 도메인 구매 및 연결
- [ ] 홈페이지 공개
- [ ] 네이버 플레이스 등록
- [ ] 학부모에게 홈페이지 오픈 안내

#### 5단계: 정착 및 루틴 (4주~)

- [ ] 주간 콘텐츠 루틴 시작 (주 2~3개)

### 3. 구글 비즈니스 프로필 등록

> 구글 지도에서 "울산 피아노학원"을 검색했을 때 나오게 하는 것!

#### 등록 방법

1. [Google 비즈니스 프로필](https://business.google.com) 접속
2. **"지금 관리하기"** 클릭
3. 학원 이름 검색 → 없으면 **"비즈니스 추가"**
4. 정보 입력:
   - 학원 이름
   - 카테고리: "피아노 학원" 또는 "음악 학교"
   - 주소 (정확하게!)
   - 전화번호
   - 영업시간
5. **소유권 확인** (엽서, 전화, 이메일 중 선택)
6. 확인 완료 후 사진, 소개글 추가

#### 등록하면 좋은 점

- 구글에서 "울산 피아노학원" 검색 시 지도에 표시
- 전화번호, 위치, 영업시간이 한눈에
- 학부모 리뷰 수집 가능
- 네이버 플레이스 + 구글 비즈니스 둘 다 하면 검색 노출 극대화

### 4. 네이버 플레이스 등록 (필수!)

> 네이버에서 "울산 피아노학원" 검색 시 지도에 나오게 하는 것!
> **네이버 블로그 운영가이드**에 자세한 방법이 있습니다.

#### 왜 중요한가?

- 학부모의 **80% 이상**이 네이버로 학원을 검색
- 네이버 플레이스에 등록되면 검색 시 **지도 + 정보가 상단에 노출**
- 블로그, 홈페이지보다 **네이버 플레이스가 가장 먼저 보임**
- 리뷰, 사진, 영업시간, 연락처가 한눈에 표시

> **결론: 네이버 플레이스 등록은 선택이 아니라 필수입니다!**

### 5. 학부모 리뷰 수집

#### 리뷰가 중요한 이유

- 새로운 학부모가 학원을 알아볼 때 **가장 먼저 보는 것이 리뷰**
- 네이버 플레이스/구글 리뷰가 많을수록 **검색 상위 노출**
- 실제 학부모의 후기가 **가장 강력한 마케팅**

#### 리뷰 요청 방법 (자연스럽게)

| 시점 | 방법 |
|------|------|
| **발표회 후** | "오늘 발표회 어떠셨어요? 네이버에 리뷰 남겨주시면 감사하겠습니다 😊" |

> **주의:** 리뷰 대가로 금전/선물을 제공하면 안 됩니다. (공정거래법)

| **입구** | QR코드 스탠드 (인스타/카톡 채널) |
| **레슨실 벽면** | 학생 연주 사진 액자 + "더 많은 영상은 유튜브에서!" |
| **대기 공간** | 학원 명함 비치 + 인스타 팔로우 안내 |
| **피아노 위** | 아크릴 스탠드 "인스타 @아이디" |

#### QR코드 만들기

1. 네이버 검색: **"QR코드 생성기"**
2. 인스타그램/카카오톡 채널 주소 입력
3. QR코드 이미지 다운로드
4. A4에 인쇄하여 학원에 비치
