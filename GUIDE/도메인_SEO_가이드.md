# 도메인 구매 & SEO 가이드

리틀버드 음악학원 홈페이지의 도메인 연결과 검색 노출을 위한 가이드입니다.

---

## 1. 도메인 구매 & 연결

### 1-1. 도메인 구매

추천 사이트:
- **가비아** (gabia.com) - 국내 1위, 관리 편리
- **호스팅케이알** (hosting.kr) - 가격이 저렴한 편

구매 순서:
1. 가비아(gabia.com) 또는 호스팅케이알(hosting.kr) 접속
2. 회원가입 및 로그인
3. 원하는 도메인 검색 (예: `littlebirdmusic.com` 또는 `littlebirdmusic.kr`)
   - `.com` : 전세계 공용, 가장 보편적
   - `.kr` : 한국 도메인, 네이버 검색에 약간 유리할 수 있음
4. 사용 가능한 도메인 선택 후 결제 (보통 연 1~2만원)
5. 결제 완료 후 "도메인 관리" 메뉴에서 DNS 설정으로 이동

### 1-2. Vercel에 도메인 연결

**DNS 설정** (가비아/호스팅케이알 관리 페이지에서):

| 타입 | 호스트 | 값 |
|------|--------|-----|
| A | @ | `76.76.21.21` |
| CNAME | www | `cname.vercel-dns-0.com` |

설정 방법:
1. 도메인 관리 페이지 → DNS 설정(또는 네임서버 설정)
2. 위 표대로 레코드 2개 추가
3. 저장

**Vercel 대시보드에서 도메인 추가**:
1. Vercel 대시보드(vercel.com) 로그인
2. 리틀버드 프로젝트 선택
3. **Settings** → **Domains** 클릭
4. 구매한 도메인 입력 (예: `littlebirdmusic.com`)
5. Add 클릭
6. DNS 설정이 반영되면 자동으로 HTTPS(보안 인증서)도 적용됨

> DNS 반영에 최대 24~48시간 걸릴 수 있지만, 보통 1~2시간이면 됩니다.

## 2. 네이버 서치어드바이저 등록

네이버에서 학원 홈페이지가 검색되게 하려면 반드시 등록해야 합니다.

### 2-1. 사이트 등록

1. https://searchadvisor.naver.com 접속
2. 네이버 로그인
3. **웹마스터 도구** 클릭
4. **사이트 추가** 클릭
5. 홈페이지 주소 입력 (예: `https://littlebirdmusic.com`)

### 2-2. 사이트 소유 확인 (인증)

두 가지 방법 중 하나 선택:

**방법 A: HTML 메타 태그 (추천)**
1. 네이버가 제공하는 메타 태그 코드 복사
2. `index.html` 파일의 `<head>` 안에 붙여넣기
   ```html
   <meta name="naver-site-verification" content="여기에_인증코드" />
   ```
3. 저장 후 배포
4. 네이버에서 "확인" 버튼 클릭

**방법 B: HTML 파일 업로드**
1. 네이버가 제공하는 HTML 파일 다운로드
2. 프로젝트 루트 폴더에 파일 저장
3. 배포 후 네이버에서 "확인" 클릭

### 2-3. sitemap.xml 제출

1. 프로젝트 루트에 `sitemap.xml` 파일 생성 (아래 내용 복사):
   ```xml
   <?xml version="1.0" encoding="UTF-8"?>
   <urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
     <url>
       <loc>https://도메인주소/</loc>
       <lastmod>2026-03-08</lastmod>
       <priority>1.0</priority>
     </url>
   </urlset>
   ```
   - `https://도메인주소/` 부분을 실제 도메인으로 변경
2. 배포
3. 네이버 서치어드바이저 → **요청** → **사이트맵 제출**
4. `https://도메인주소/sitemap.xml` 입력 후 제출

### 2-4. robots.txt 설정

프로젝트 루트에 `robots.txt` 파일 생성:
```
User-agent: *
Allow: /
Sitemap: https://도메인주소/sitemap.xml
```
- `도메인주소` 부분을 실제 도메인으로 변경
- 배포

## 3. 구글 Search Console 등록

구글 검색에 노출되려면 등록이 필요합니다.

### 3-1. 속성 추가

1. https://search.google.com/search-console 접속
2. 구글 계정 로그인
3. **속성 추가** 클릭
4. **URL 접두어** 선택
5. 홈페이지 주소 입력 (예: `https://littlebirdmusic.com`)

### 3-2. 소유권 확인

**방법 A: HTML 태그 (추천)**
1. 구글이 제공하는 메타 태그 복사
2. `index.html`의 `<head>` 안에 붙여넣기:
   ```html
   <meta name="google-site-verification" content="여기에_인증코드" />
   ```
3. 저장 후 배포
4. 구글에서 "확인" 클릭

**방법 B: DNS 인증**
1. 구글이 제공하는 TXT 레코드 복사
2. 도메인 관리(가비아/호스팅케이알) → DNS 설정
3. TXT 레코드 추가
4. 구글에서 "확인" 클릭

### 3-3. sitemap 제출

1. Search Console 좌측 → **Sitemaps**
2. `https://도메인주소/sitemap.xml` 입력
3. 제출

## 4. 네이버 플레이스 최적화

울산 지역 학원이므로 네이버 플레이스가 매우 중요합니다. "울산 피아노학원" 검색 시 지도와 함께 노출됩니다.

### 4-1. 네이버 플레이스 등록/수정

1. **스마트플레이스 센터** 접속: https://smartplace.naver.com
2. 네이버 로그인
3. "리틀버드 음악학원" 검색 → 이미 등록되어 있으면 "소유자 인증" 진행
4. 등록되어 있지 않으면 **새 업체 등록**

### 4-2. 정보 완전히 채우기

다음 항목을 빠짐없이 입력:
- **업체명**: 리틀버드 음악학원
- **카테고리**: 피아노학원 / 음악학원
- **주소**: 정확한 주소 입력
- **영업시간**: 월~금 13:30~18:30, 토/일/공휴일 휴무
- **전화번호**: 학원 연락처
- **소개글**: 학원 특징 200자 내외로 작성
  - 예: "울산 리틀버드 음악학원은 초등학생 맞춤 피아노 교육을 합니다. 기초부터 재즈, 합주, 연주회까지 즐거운 음악 경험과 탄탄한 기초 교육을 제공합니다."
- **사진**: 학원 외부, 내부, 레슨실, 연주회 사진 등 5장 이상
- **편의정보**: 주차 가능 여부, 와이파이 등

### 4-3. 키워드 전략

소개글과 메뉴/서비스에 아래 키워드를 자연스럽게 포함:
- "울산 피아노학원"
- "울산 음악학원"
- "울산 [동네명] 피아노" (예: "울산 중구 피아노학원")
- "초등학생 피아노"
- "어린이 피아노"

### 4-4. 리뷰 관리

- 수업 후 학부모에게 네이버 리뷰 부탁하기
  - "네이버에서 '리틀버드 음악학원' 검색 → 리뷰 작성"
- 모든 리뷰에 감사 답글 달기 (성의 있게)
- 부정적 리뷰에도 정중하게 답변

### 4-5. 소식 기능 활용

- 스마트플레이스 → **소식** 탭
- 정기적으로 소식 업데이트 (월 2~4회 권장)
  - 연주회 안내, 새 특강 개설, 수강생 소식 등
- 사진과 함께 올리면 효과 좋음

### 4-6. 예약 기능

- 스마트플레이스 → **예약** 설정
- 체험 레슨 예약을 받으면 신규 문의 전환에 도움

## 5. 울산 지역 SEO 전략

### 5-1. 홈페이지 메타 태그 확인

`index.html`의 `<meta name="description">`에 "울산", "피아노학원" 키워드가 포함되어 있는지 확인:
```html
<meta name="description" content="울산 리틀버드 음악학원 - 아이의 첫 번째 피아노 선생님...">
```
- 이미 포함되어 있다면 OK

### 5-2. 네이버 블로그 연동

- 기존 `네이버블로그_운영가이드.md` 참조
- 블로그 글에 홈페이지 링크 포함
- 블로그 프로필에 홈페이지 주소 등록

### 5-3. 카카오맵 등록

1. https://place.map.kakao.com 접속
2. 카카오 로그인
3. "리틀버드 음악학원" 검색
4. 이미 등록되어 있으면 "정보 수정 제안" → 소유자 인증
5. 없으면 **새 장소 등록**
6. 정보 입력: 주소, 전화번호, 영업시간, 사진

### 5-4. 콘텐츠에 지역 키워드 포함

홈페이지, 블로그, SNS에 아래 키워드를 자연스럽게 포함:
- "울산 피아노학원"
- "울산 음악학원"
- "울산 [동네명] 피아노"
- "울산 초등학생 피아노"
- "울산 어린이 음악교육"

> 키워드를 억지로 반복하지 말고, 글 흐름에 자연스럽게 녹여주세요.
