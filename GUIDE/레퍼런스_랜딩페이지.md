## 레퍼런스 목록

### 1. Calm
- **URL**: https://www.calm.com
- **장점**: 극도의 미니멀리즘, 차분한 블루 컬러, 감성적 배경 이미지, 짧은 카피와 단일 CTA
- **적용 포인트**: 시각적 안정감과 단순한 구조. 색상 톤과 여백 활용 참고

### 2. Slack
- **URL**: https://slack.com
- **장점**: 일러스트레이션으로 복잡한 개념을 시각적으로 설명, 밝은 색상 체계, 단계별 기능 소개 섹션
- **적용 포인트**: SNS 운영 플로우를 일러스트/아이콘 기반으로 단계별 안내하는 구조. "How it works" 섹션 패턴

### 3. Stripe Atlas (가이드 페이지)
- **URL**: https://stripe.com/atlas
- **장점**: 단계별 가이드 구조(Step 1→2→3), 깔끔한 타이포그래피, 아이콘과 텍스트의 밸런스, 신뢰감 있는 디자인
- **적용 포인트**: "단계별 안내"에 직접 적용 가능한 레이아웃 패턴. 프로세스 시각화

### 4. Framer 블로그 - Landing Page Best Practices
- **URL**: https://www.framer.com/blog/landing-page-best-practices/
- **장점**: F/Z 패턴 레이아웃, 스토리텔링 기법, 실제 적용 사례 포함. 최신 트렌드 반영
- **적용 포인트**: 전체 페이지 구조 설계. 특히 "5초 테스트"(5초 안에 가치 전달) 원칙

### 5. One Page Love (원페이지 디자인 갤러리)
- **URL**: https://onepagelove.com
- **장점**: 8,000+ 원페이지 웹사이트 모음, 태그 기반 필터링, 가이드/튜토리얼 유형 별도 카테고리
- **적용 포인트**: "guide" 또는 "how-to" 태그로 필터링하면 유사한 원페이지 안내 사이트 다수

### 6. Lapa Ninja (미니멀 카테고리)
- **URL**: https://www.lapa.ninja/category/minimal/
- **장점**: 7,300+ 랜딩페이지, 미니멀 카테고리에 3,049개 디자인. 풀페이지 스크린샷 + 비디오 레코딩 제공
- **적용 포인트**: 미니멀하면서도 정보 전달력 있는 레이아웃. 컬러/폰트 조합 참고

### 7. JoinClyde
- **URL**: https://joinclyde.com
- **장점**: 스마트 일러스트 + 소셜 프루프, 섹션마다 CTA로 자연스럽게 유도
- **적용 포인트**: 일러스트와 아이콘을 활용한 단계별 안내 + 섹션 전환의 모멘텀 구축 방식

### 8. Sprout Social - SNS 랜딩페이지 가이드
- **URL**: https://sproutsocial.com/insights/landing-page-examples/
- **장점**: SNS 플랫폼별 랜딩페이지 15개 실제 사례. 모바일 우선, 일관된 메시징, 비주얼 계층 구조 분석
- **적용 포인트**: SNS 맥락에 맞는 랜딩페이지 디자인 패턴

### 9. Anime.js
- **URL**: https://animejs.com
- **장점**: 인터랙티브 스크롤 애니메이션이 돋보이는 단일 페이지. 기술 가이드를 시각적으로 표현
- **적용 포인트**: 스크롤 기반 콘텐츠 공개 방식. 단계를 스크롤로 하나씩 보여주는 인터랙션

### 10. Netflix
- **URL**: https://www.netflix.com
- **장점**: 미니멀리즘의 교과서. 다크 배경 + 볼드 타이포 + 단일 CTA. 200단어 미만의 카피로 가치 전달
- **적용 포인트**: "less is more" 원칙. 섹션별 한 가지 메시지만 전달하는 구조

## 적용 종합 인사이트

| 관점 | 참고 패턴 | 레퍼런스 |
|------|----------|----------|
| 구조 | Step-by-step 프로세스 가이드 + 스크롤 기반 | Stripe Atlas, Anime.js |
| 비주얼 | 커스텀 일러스트/아이콘 + 미니멀 레이아웃 | Slack, JoinClyde |
| 인터랙션 | 스크롤 애니메이션으로 순차 공개 | Anime.js |
| 컬러 | 2~3색 체계, 1개 액센트 컬러로 CTA 강조 | Calm, Netflix |
| 모바일 | 엄지 터치 친화, 3초 이내 첫 콘텐츠 | Sprout Social |
| 카피 | 250~300단어 이하, 헤드라인 가치 전달, 불릿 포인트 | Netflix, Framer |
