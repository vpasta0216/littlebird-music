## 1. 인스타그램 비즈니스 계정 만들기

### 1단계: 계정 생성

1. 앱스토어에서 **Instagram** 설치
2. **"새 계정 만들기"** 클릭
3. 이메일 주소 입력 (학원 전용 이메일 추천)
4. 사용자 이름 설정

### 사용자 이름 추천

| 유형 | 예시 |
|------|------|
| **학원명 영문** | @doremi_piano |
| **학원명+지역** | @doremi_piano_ulsan |
| **학원명+music** | @doremi.music |

> 짧고 기억하기 쉬운 이름이 좋아요. 특수문자는 언더바(_)와 마침표(.)만 가능!

### 2단계: 비즈니스 계정 전환

1. 프로필 > 오른쪽 위 **≡** (메뉴)
2. **설정 및 활동** > **계정 유형 및 도구**
3. **"프로페셔널 계정으로 전환"** 클릭
4. 카테고리: **"음악 교습"** 또는 **"교육"** 선택
5. **"비즈니스"** 선택 (크리에이터 아님!)
6. 연락처 정보 입력 (전화번호, 이메일, 주소)

> **왜 비즈니스 계정?**
> - 게시물 성과 통계 확인 가능
> - 프로필에 "전화하기", "길찾기" 버튼 추가
> - 광고 집행 가능 (나중에 필요하면)
> - 인사이트(분석) 기능 사용 가능

## 4. 릴스(Reels) 가이드

> 릴스 = 인스타그램의 짧은 세로 영상 (유튜브 쇼츠와 동일!)

### 릴스 올리는 방법

1. 인스타그램 앱 > 하단 **[+]** 버튼
2. **"릴스"** 선택
3. 갤러리에서 영상 선택 (또는 바로 촬영)
4. 길이 조절 (15~30초 추천)
5. 음악 추가 (선택사항 - 피아노 연주면 불필요)
6. 자막/텍스트 추가
7. **"다음"** > 캡션 작성 > **"공유"**

### 릴스 vs 일반 게시물

| | 일반 게시물 | 릴스 |
|---|-----------|------|
| **형태** | 사진 또는 짧은 영상 | 세로 영상 (최대 90초) |
| **노출** | 팔로워 위주 | 비팔로워에게도 많이 노출 |
| **조회수** | 보통 | 높음 (10배 이상 차이!) |
| **추천** | 사진 콘텐츠 | 영상 콘텐츠 |

> **핵심:** 연주 영상은 무조건 **릴스**로 올리세요! 일반 게시물보다 조회수가 훨씬 많습니다.

### 유튜브 쇼츠 → 인스타 릴스 동시 활용

1. 영상을 **세로로 촬영**
2. **CapCut**으로 편집 (자막 포함)
3. 편집된 영상을 **핸드폰에 저장**
4. **유튜브 쇼츠**로 업로드
5. **같은 영상**을 인스타 릴스로도 업로드

> 한 번 만들면 두 곳에 올릴 수 있어요!

## 5. 스토리 활용법

### 스토리란?

- 24시간 후 사라지는 콘텐츠
- 부담 없이 올릴 수 있음
- 팔로워와 소통하기 좋음

### 스토리 활용 아이디어

| 유형 | 예시 |
|------|------|
| **오늘의 수업** | 수업 중 짧은 영상 (10초) |
| **연습 독려** | "오늘 연습 했나요?" 투표 스티커 |
| **곡 맞추기** | 연주 3초 → "무슨 곡일까요?" 퀴즈 |
| **일상 공유** | 출근길, 수업 준비, 간식 타임 |
| **공지 알림** | "내일 수업 시간 변경!" |
| **게시물 공유** | 새 게시물 올렸을 때 스토리에도 공유 |

### 스토리 하이라이트

> 스토리는 24시간 후 사라지지만, **하이라이트**에 저장하면 프로필에 계속 남아요!

**추천 하이라이트 구성:**

| 하이라이트 | 아이콘 | 내용 |
|------------|--------|------|
| **학원소개** | 🏠 | 학원 시설, 위치, 분위기 |
| **학생연주** | 🎹 | 학생 연주 베스트 모음 |
| **수업모습** | 📚 | 수업 풍경, 교재 소개 |
| **발표회** | 🎭 | 발표회 하이라이트 |
| **후기** | 💛 | 학부모 후기 캡쳐 |

## 8. Canva로 예쁜 게시물 만들기

> **Canva(캔바)**는 무료 디자인 앱이에요. 디자인 못해도 템플릿만 바꾸면 됩니다!

### 사용 방법

1. 앱스토어에서 **Canva** 설치
2. 무료 회원가입
3. **"인스타그램 게시물"** 검색
4. 마음에 드는 **템플릿** 선택
5. 텍스트, 사진만 바꾸기
6. **다운로드** > 인스타에 업로드

### 추천 활용

## 9. 인스타그램 운영 루틴

### 주간 루틴

| 요일 | 할 일 |
|------|-------|
| **월** | 이번 주 올릴 콘텐츠 정하기 (3개) |
| **화** | 학생 연주 영상 촬영 + 편집 |
| **수** | 연주 영상 릴스 업로드 + 스토리 |
| **목** | 학원 일상 사진 촬영 |
| **금** | 일상 게시물 업로드 + 스토리 |
| **토** | 다음 주 콘텐츠 아이디어 메모 |
| **일** | 쉬기! (자동 예약 게시물 활용) |

### 일일 루틴 (5분)

1. 스토리 1개 올리기 (수업 중 짧은 영상이나 사진)
2. 댓글/DM 확인하고 답변
3. 다른 학원이나 음악 계정 1~2개 소통 (좋아요, 댓글)

### 월간 루틴

- [ ] 이번 달 인사이트(통계) 확인
- [ ] 가장 반응 좋았던 게시물 유형 파악
- [ ] 다음 달 콘텐츠 방향 조정
- [ ] 프로필 정보 업데이트 필요한지 확인

## 10. 팔로워 늘리는 방법

### 자연스럽게 팔로워 늘리기

| 방법 | 설명 |
|------|------|
| **꾸준한 업로드** | 주 3회 이상이 이상적 (최소 주 1회) |
| **릴스 활용** | 릴스는 비팔로워에게 노출되어 새 팔로워 유입 |
| **해시태그** | 지역 해시태그로 동네 학부모에게 도달 |
| **소통** | 댓글에 답글 달기, DM 빠르게 응대 |
| **다른 계정 소통** | 울산 맘카페, 학부모 계정에 좋아요+댓글 |
| **오프라인 연결** | 학원에 "인스타 팔로우하면 스티커 증정" |
| **학부모 공유** | "오늘 올린 영상 확인해보세요~" 문자 |

### 하지 말아야 할 것

- 팔로워 구매 (가짜 계정, 의미 없음)
- 무차별 팔로우/언팔로우
- 관련 없는 해시태그 남용
- DM으로 광고 메시지 보내기
- 다른 계정 게시물에 학원 홍보 댓글
