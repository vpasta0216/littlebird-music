# 카카오톡 채널 만들기 & 자동응답 설정 가이드

> 이 문서는 카카오톡 채널을 처음 만들고,
> 자동응답 메시지와 알림을 설정하는 방법을 안내합니다.
> 순서대로 따라하시면 됩니다!

---

## STEP 1. 카카오톡 채널 만들기

카카오톡 채널은 **무료**로 만들 수 있습니다! 컴퓨터나 핸드폰 둘 다 가능하지만, 컴퓨터가 더 편합니다.

### 컴퓨터에서 만들기

1. 인터넷 브라우저에서 이동: **https://business.kakao.com/**
2. 오른쪽 위 **"카카오 로그인"** 클릭 → 누나의 카카오톡 계정으로 로그인
3. 로그인 후 **"카카오톡 채널"** 메뉴를 클릭
4. **"새 채널 만들기"** 버튼을 클릭
5. 아래 정보를 입력:

| 항목 | 입력 내용 |
|------|-----------|
| **채널 이름** | 학원 이름 그대로 입력 (예: 도레미피아노학원) |
| **검색용 아이디** | 영어로 짧게 만들기 (예: doremi-piano) |
| **프로필 사진** | 학원 로고 또는 학원 사진 (나중에 바꿀 수 있음) |
| **소개글** | 학원 한 줄 소개 (예: "울산 ○○아파트 상가 피아노학원") |
| **카테고리** | 교육 > 음악/미술학원 |

6. **"채널 만들기"** 버튼 클릭하면 완성!

### 확인하기

만들어진 후 카카오톡 앱에서 확인:
> 카카오톡 > 더보기(···) > 카카오톡 채널 > 내 채널 관리
> 여기서 방금 만든 채널이 보이면 성공!

## STEP 2. 채널 프로필 꾸미기

학부모님들이 처음 보는 화면이니 중요합니다!

1. **카카오톡 채널 관리자센터** 접속: **https://center-pf.kakao.com**
2. 카카오 로그인 → 만든 채널 선택
3. 왼쪽 메뉴에서 **"관리 > 상세설정"** 클릭
4. 아래 항목들을 채워주세요:

| 항목 | 내용 |
|------|------|
| **프로필 사진** | 학원 로고 또는 깔끔한 학원 사진 (정사각형, 640x640 이상) |
| **배경 사진** | 학원 내부 사진이나 피아노 사진 (960x540 이상) |
| **운영시간** | 월~금 14:00~21:00 / 토 10:00~17:00 |
| **전화번호** | 학원 전화번호 |
| **주소** | 학원 주소 |

**소개글 추천 문구:**

```
안녕하세요, ○○피아노학원입니다.
울산 ○○아파트 상가에서 4년째
아이들의 피아노 교육을 함께하고 있습니다.
기초부터 탄탄하게, 아이의 속도에 맞춰
즐겁게 배울 수 있어요.
수업 문의는 채팅으로 편하게 해주세요!
```

5. **"저장"** 버튼 클릭

## STEP 3. 자동응답 메시지 설정 (1차 자동응답)

> **이 부분이 가장 중요합니다!**
> 수업 중에 바로 답장 못하니까, 문의가 오면 자동으로 답변이 가도록 설정합니다.

1. 카카오톡 채널 관리자센터에서 왼쪽 메뉴 **"1:1채팅 > 채팅방 자동응답"** 클릭
2. **"채팅방 자동응답 사용"** 을 **켜기(ON)** 로 바꿈
3. **"시작 메시지"** 설정하기 (학부모가 처음 채팅 시작 시 자동으로 나가는 메시지)

**추천 시작 메시지:**

```
안녕하세요! 🎵 ○○피아노학원입니다.

문의해 주셔서 감사합니다!
수업 중에는 답변이 늦을 수 있지만,
확인 후 빠르게 답변드리겠습니다.

빠른 상담을 원하시면 아래 내용을 남겨주세요:

✏ 학생 이름:
✏ 학생 나이/학년:
✏ 연락처:
✏ 문의 내용:

감사합니다! 😊
```

> 위 문구를 그대로 복사해서 사용하셔도 되고, 학원 이름만 바꿔주세요!

## STEP 4. 영업시간 외 자동안내 메시지 설정

밤이나 일요일에 문의가 오면 자동으로 안내 메시지가 가도록 설정합니다.

1. 관리자센터에서 왼쪽 메뉴 **"1:1채팅 > 운영시간 설정"** 클릭
2. **운영시간 설정:**

| 요일 | 시간 |
|------|------|
| 월~금 | 14:00 ~ 21:00 |
| 토 | 10:00 ~ 17:00 |
| 일 | OFF (휴무) |

> 실제 학원 운영시간에 맞게 바꿔주세요!

3. **"운영시간 외 자동 메시지"** 를 **켜기(ON)**
4. 메시지 작성:

**추천 메시지:**

```
안녕하세요, ○○피아노학원입니다.

지금은 운영시간이 아니라 바로 답변이 어렵습니다.

🕐 운영시간: 월~금 14:00~21:00 / 토 10:00~17:00

운영시간에 확인 후 빠르게 답변드리겠습니다!
급하신 경우 전화 주세요: 052-○○○-○○○○

문의 내용을 미리 남겨주시면 더 빠른 상담이 가능합니다:

✏ 학생 이름:
✏ 학생 나이/학년:
✏ 연락처:
✏ 문의 내용:

감사합니다! 😊
```

5. **"저장"** 클릭

## STEP 5. 알림 설정 (문의가 오면 카카오톡으로 알림받기)

문의가 들어오면 누나의 핸드폰 카카오톡으로 알림이 오도록 설정합니다.

### 핸드폰 앱에서 설정

1. 핸드폰에서 **"카카오톡 채널 관리자"** 앱을 설치

| 기기 | 설치 방법 |
|------|-----------|
| **아이폰** | App Store에서 "카카오톡 채널 관리자" 검색 |
| **안드로이드** | Play Store에서 "카카오톡 채널 관리자" 검색 |

2. 앱을 열고 카카오 계정으로 로그인
3. 만든 채널을 선택
4. 이제부터 **문의가 오면 핸드폰에 푸시 알림**이 옵니다! 앱에서 바로 답장 가능!

### 알림이 안 올 때 확인사항

- 핸드폰 설정 > 알림 > "카카오톡 채널 관리자" > 알림 허용 되어있는지
- 앱 안에서 채널 선택 > 설정(톱니바퀴) > 알림 설정 > ON 되어있는지
- 핸드폰이 "방해금지 모드"는 아닌지

### 여러 명이 관리하고 싶을 때

동생도 같이 알림을 받고 답장하고 싶다면:

1. 카카오톡 채널 관리자센터 (컴퓨터) 접속
2. 왼쪽 메뉴 **"관리 > 관리자"** 클릭
3. **"관리자 초대"** 클릭
4. 동생의 카카오톡 아이디를 검색해서 초대
5. 동생도 "카카오톡 채널 관리자" 앱을 설치하면 끝!

> 이렇게 하면 누나가 수업 중일 때 동생이 대신 답장할 수 있어요!

## STEP 6. 자주 묻는 질문 자동응답 (선택사항)

> 이건 선택사항입니다! 나중에 해도 되지만,
> 설정하면 반복 질문에 일일이 답하지 않아도 돼서 편합니다.

1. 관리자센터에서 왼쪽 메뉴 **"채팅 > 시작 버튼"** 클릭
2. 버튼을 추가합니다:

### 버튼 1: 수업 안내

```
○○피아노학원 수업 안내

🎹 수업 구성
- 1:1 개인 레슨 (주 2~3회)
- 1회 수업시간: 40분
- 대상: 유치원~초등학생

🎹 수업 과정
입문 → 초급 → 중급 → 심화
아이의 수준에 맞춰 진행합니다.

자세한 상담은 채팅으로 문의해주세요!
```

### 버튼 2: 수강료 문의

```
수강료는 수업 횟수와 시간에 따라 다릅니다.

정확한 안내를 위해 아래 정보를 남겨주시면
맞춤 상담을 도와드리겠습니다!

✏ 학생 이름:
✏ 학생 나이/학년:
✏ 피아노 경험: 있음( 개월) / 없음
✏ 원하는 수업 횟수: 주 2회 / 주 3회
✏ 연락처:
```

### 버튼 3: 위치 & 운영시간

```
○○피아노학원 위치 안내

📍 주소: 울산광역시 ○○구 ○○동 ○○아파트 상가 ○층
🚗 주차: 상가 건물 주차장 이용 가능

🕐 운영시간
월~금: 14:00~21:00
토: 10:00~17:00
일: 휴무

📞 전화: 052-○○○-○○○○
```

### 버튼 4: 체험 레슨 신청

```
무료 체험 레슨을 신청해주셔서 감사합니다! 🎹

아래 정보를 남겨주시면
가능한 시간을 안내해드리겠습니다.

✏ 학생 이름:
✏ 학생 나이/학년:
✏ 피아노 경험: 있음 / 없음
✏ 희망 요일/시간:
✏ 연락처:

체험 레슨은 약 30분 진행됩니다.
```

3. **"저장"** 클릭하면 완성!

## STEP 7. 홈페이지에 카카오톡 상담 버튼 넣기

> 이 부분은 제가 해드립니다!

카카오톡 채널이 만들어지면 아래 정보만 알려주세요:

1. **카카오톡 채널 검색용 아이디** (STEP 1에서 만든 영어 아이디) - 예) @doremi-piano
2. **채널 URL** (관리자센터 > 관리 > 상세설정에서 확인) - 예) https://pf.kakao.com/_xxxxx

이 정보를 주시면 홈페이지에:
- 우측 하단 카카오톡 상담 버튼
- SNS 배너에 카카오톡 채널 링크
- 수강문의 페이지에 "카카오톡으로 문의하기" 버튼

을 넣어드리겠습니다!

## STEP 8. 홈페이지 수강문의 → 카카오톡 알림 연결

### 방법 A: 간단한 방법 (이메일 + 카카오톡 관리자 앱)

1. 홈페이지 문의 폼 → 이메일로 전달되도록 설정 (제가 해드림)
2. 핸드폰 이메일 앱에서 알림 ON
3. 카카오톡 채널 관리자 앱에서 직접 문의도 확인

> 이메일 + 카카오톡 두 곳에서 문의를 확인할 수 있어요!

### 방법 B: 고급 방법 (카카오톡 직접 알림)

홈페이지 문의가 오면 → 카카오톡으로 바로 알림. 약간의 추가 설정이 필요합니다. 제가 설정해드릴 수 있으니 나중에 말씀해주세요!

## 해야 할 일 체크리스트

- [ ] 1. 카카오톡 채널 만들기 (STEP 1) → https://business.kakao.com 에서 10분이면 완성
- [ ] 2. 프로필 꾸미기 (STEP 2) → 학원 사진, 소개글, 운영시간 입력
- [ ] 3. 자동응답 설정 (STEP 3) → 위의 추천 문구 복사 + 학원 정보만 수정
- [ ] 4. 영업시간 외 자동안내 설정 (STEP 4) → 운영시간 입력 + 추천 문구 복사
- [ ] 5. 핸드폰에 "카카오톡 채널 관리자" 앱 설치 (STEP 5) → 알림 켜기!
- [ ] 6. (선택) 자주 묻는 질문 버튼 설정 (STEP 6)
- [ ] 7. 채널 정보를 알려주기 (STEP 7) → 채널 아이디, URL을 알려주시면 홈페이지에 연결

> **1~5번까지 하면 기본 설정 완료! 소요시간: 약 30분~1시간**
