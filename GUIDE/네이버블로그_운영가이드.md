> 네이버 블로그는 한국에서 가장 강력한 마케팅 채널입니다!
> 학부모님들이 "울산 피아노학원" 검색하면 네이버 블로그가 상위에 노출됩니다.

## STEP 1. 네이버 블로그 만들기

> 네이버 아이디가 있으면 블로그도 이미 있습니다!

### 네이버 아이디가 없는 경우

1. **https://nid.naver.com/user2/join** 접속
2. 약관 동의 → 본인 인증 → 아이디/비밀번호 설정
3. 완성!

> **팁:** 아이디를 학원 관련으로 만드세요.
> 예) `doremi_piano`, `piano_ulsan`
> 이 아이디가 블로그 주소가 됩니다: `blog.naver.com/doremi_piano`

### 블로그 시작하기

1. **https://blog.naver.com** 접속
2. 네이버 로그인
3. **"블로그 시작하기"** 클릭 (처음만)
4. 블로그 이름 입력 (예: "○○피아노학원 | 울산 피아노 레슨")
5. 별명 입력 (예: "○○피아노학원")
6. 완성!

## STEP 3. 글 쓰는 방법

### 글 쓰기 기본 순서

1. 블로그 접속 > 왼쪽 **"글쓰기"** 클릭
2. **제목** 입력
3. **카테고리** 선택 (오른쪽 설정에서)
4. **본문** 작성 (아래 양식 참고)
5. **사진/영상** 넣기
6. **태그** 입력
7. **"발행"** 클릭

> **핸드폰에서도 가능!** 앱스토어에서 "네이버 블로그" 검색 후 설치. 사진/영상을 바로 올릴 수 있어서 편해요.

## STEP 5. 사진과 영상 넣는 방법

### 컴퓨터에서

1. 글쓰기 화면에서 본문 클릭
2. 상단 도구 모음에서:
   - 📷 **사진 아이콘** 클릭 → 사진 선택 → 삽입
   - 🎬 **동영상 아이콘** 클릭 → 영상 선택 → 삽입
   - 유튜브 영상: 도구 모음 > "링크" 또는 "동영상" > 유튜브 URL 붙여넣기

### 핸드폰에서

1. 네이버 블로그 앱 > 글쓰기
2. 📷 아이콘 클릭 → 갤러리에서 선택 → 바로 삽입

### 사진 잘 넣는 팁

```
글 (2~3줄)
  ↓
사진 1장
  ↓
글 (2~3줄)
  ↓
사진 1~2장
  ↓
글 (마무리)
```

> **글만 길게 쓰면 아무도 안 읽어요!** 사진이 중간중간 있어야 스크롤하면서 읽게 됩니다.

> **팁:** 영상은 블로그에 직접 올리는 것보다 **유튜브에 올리고 블로그에 삽입**하는 게 좋아요.
> (유튜브 조회수도 올라가고, 블로그 용량도 안 차니까)

5. **사업자등록증 사진** 첨부 (인증용)

## STEP 8. 블로그 운영 루틴

### 주 1~2회 루틴 (약 20~30분)

1. 이번 주 촬영한 사진/영상 중 좋은 것 고르기
2. 위 양식 중 하나 골라서 복사
3. 내용 채우기 (사진 삽입 + 간단한 글)
4. 태그 입력
5. **"발행"** 클릭 → 끝!

### 추천 월간 콘텐츠 계획

| 주차 | 콘텐츠 |
|------|--------|
| **1주차** | 학생 연주 영상 게시물 |
| **2주차** | 학원 일상 / 소식 |
| **3주차** | 학생 연주 영상 게시물 |
| **4주차** | 피아노 팁 / 교육 정보 |
| **발표회 달** | 발표회 후기 추가 |
| **모집 시기** | 수강생 모집 안내 글 추가 |

### 처음 시작할 때 올려야 할 글 (순서대로)

| 순서 | 글 내용 | 양식 |
|------|---------|------|
| **1번째** | 학원 소개 (가장 먼저!) | 양식 4 |
| **2번째** | 수업/커리큘럼 안내 | - |
| **3번째** | 오시는 길 / 위치 안내 | - |
| **4번째** | 학생 연주 영상 (가장 잘 된 것) | 양식 1 |
| **5번째** | 학원 분위기 / 시설 소개 | - |

> **이 5개를 먼저 올리면 블로그 기본 구색이 갖춰집니다!**

## STEP 9. 블로그 글 → 인스타/유튜브 연결하기

> **"한 번 만들고 여러 번 쓰기"**가 핵심!

### 콘텐츠 재활용 흐름

학생 연주를 촬영했다면:

| 순서 | 채널 | 내용 |
|------|------|------|
| 1 | **유튜브** | 전체 연주 영상 업로드 (가로, 2~5분) |
| 2 | **유튜브 쇼츠** | 하이라이트 15~30초 (세로) |
| 3 | **인스타 릴스** | 같은 쇼츠 영상 올리기 (세로) |
| 4 | **인스타 게시물** | 연주 사진 + 짧은 소개 |
| 5 | **네이버 블로그** | 양식 1로 글 작성 + 유튜브 영상 링크 포함 |

> **하나의 촬영으로 5개의 콘텐츠 완성!**

### 블로그에 유튜브 영상 넣는 방법

1. 유튜브에서 영상 주소(URL) 복사
2. 블로그 글쓰기 > 도구 모음 > **"동영상"** 클릭
3. URL 붙여넣기 > **"확인"**
4. 영상이 블로그 본문에 삽입!

## STEP 10. 흔한 실수 & 주의사항

### ❌ 하면 안 되는 것

1. **다른 블로그 글 복사해서 올리기** → 네이버가 "저품질" 블로그로 판정, 검색에서 사라짐!
2. **한꺼번에 10개 올리고 한 달 쉬기** → 꾸준히 1~2개씩이 훨씬 효과적
3. **제목에 키워드 안 넣기** → "오늘의 일상"이면 검색 안 됨
4. **사진 없이 글만 올리기** → 최소 3~5장은 넣어주세요
5. **학생 사진 동의 없이 올리기** → 학부모 동의를 반드시 받으세요!

### ⭕ 하면 좋은 것

1. 글 마지막에 항상 **연락처** 넣기
2. 다른 학부모 블로그 **이웃 맺기** (서로이웃)
3. 댓글에 **친절하게 답변**하기
4. 사진을 **밝고 깨끗하게** 촬영
5. 학원 이름을 **제목에 꼭** 넣기
