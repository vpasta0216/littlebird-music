## 1개 촬영 → 5곳 활용 가이드

> 한 번 촬영하면 여러 플랫폼에 활용할 수 있어요!

```
📹 학생 연주 영상 1개 촬영 (세로, 1~3분)
    │
    ├─→ 🎬 유튜브: 전체 영상 업로드
    │
    ├─→ 🎬 유튜브 쇼츠: 하이라이트 15~30초 편집
    │
    ├─→ 📱 인스타 릴스: 같은 쇼츠 영상 업로드
    │
    ├─→ 📷 인스타 피드: 썸네일 캡쳐 + 감상평
    │
    └─→ 📝 네이버 블로그: 영상 + 학생 성장기 글
```

## 콘텐츠 제작 소요 시간 가이드

> 처음에는 오래 걸리지만, 익숙해지면 빨라져요!

| 콘텐츠 유형 | 촬영 | 편집 | 업로드 | 총 시간 |
|-------------|------|------|--------|---------|
| 연주 릴스/쇼츠 | 5분 | 10~15분 | 5분 | **약 20~25분** |
| 일상 피드 | 2분 | 5분 | 5분 | **약 12분** |
| 정보 카드뉴스 | - | 15~20분 (Canva) | 5분 | **약 20~25분** |

| 블로그 글 | - | 20~30분 (글쓰기) | 5분 | **약 25~35분** |
