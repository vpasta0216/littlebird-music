# 모바일 가독성 + SNS 가이드 + 도메인/SEO

> Appetite: S | 작성: 2026-03-08

---

## 1. 요구사항

### A. 홈페이지 모바일 가독성 수정
- 글자/글이 모바일에서 가독성 있게 보이도록 CSS 수정
- 현재 문제: body font-weight 300(얇음), 태그 0.75rem(작음), 긴 문단, 중간 breakpoint 부족

### B. SNS 통합 가이드 — "한눈에 들어오는 매핑 지도"
- 대상: 누나 (SNS 초보)
- 형태: 아이패드/휴대폰으로 보거나 출력 가능
- 핵심: 채널 간 연결 흐름도 (콘텐츠가 어디서 어디로 흘러가는지)
- 포함 내용:
  - 홈페이지 / 인스타 / 카카오톡 / 유튜브 각각 역할
  - 채널 간 연결: 영상 촬영 → 유튜브 업로드 → 인스타 릴스 / 인스타 포스트 → 카카오채널 알림
  - 시나리오: 일상 클립, 신학기/방학 포스터, 발표회 영상
  - 실전 방법: 릴스 만드는 법, 유튜브 업로드, 쇼츠 만드는 법, 카카오채널 활용

### C. 도메인 구매 + 검색 노출 (SEO)
- 커스텀 도메인 구매 방법 + Vercel 연결
- 네이버/구글에 홈페이지 노출되게 하는 방법

## 3. 작업 분할

### Task A: 홈페이지 모바일 가독성 수정 (~5분)
- **A1**: body font-weight 300 → 400 (모바일에서 가독성)
- **A2**: 모바일(768px 이하) 추가 스타일:
  - 본문 font-size 확대 (0.95rem → 1rem)
  - 태그 뱃지 font-size 0.75rem → 0.8rem
  - about 섹션 설명 줄간격 확대
  - hero 768px breakpoint에서 font-size 조정 (3.5rem → 2.5rem)
  - 커리큘럼 카드 min-width 300px → 280px (360px 기기 대응)
- **A3**: 인라인 style에서 font-size 지정된 곳 모바일 override 추가

### Task B: SNS 통합 가이드 — 채널 연결 흐름도 (~15분)
기존 guidebook.html을 **확장**하거나, 별도 `sns-flowmap.html` 생성.
- **B1**: 채널 역할 정의 카드 (홈페이지/인스타/카카오/유튜브 각각 한 줄 역할)
- **B2**: 콘텐츠 흐름 매핑 지도 (HTML/CSS 다이어그램)
  - 영상 촬영 → 편집 → 유튜브(긴 영상) + 인스타 릴스(짧은 클립) + 유튜브 쇼츠
  - 사진/포스터 → 인스타 피드 → 카카오채널 알림
  - 이벤트(신학기/방학/발표회) → 포스터 → 인스타 + 카카오 알림 + 홈페이지
- **B3**: 시나리오별 체크리스트 (일상 클립 / 이벤트 / 발표회)
- **B4**: 실전 방법 요약 (기존 가이드에서 핵심만 추출)
  - 릴스 올리는 법 (3단계)
  - 유튜브 업로드 (3단계)
  - 쇼츠 만드는 법 (3단계)
  - 카카오채널 알림 보내기 (3단계)
- **형태**: 단일 HTML 파일 (모바일 최적화, 인쇄 가능)

### Task C: 도메인 + SEO 가이드 (~5분)
- **C1**: 도메인 구매 방법 (가비아/호스팅케이알 등) + Vercel 연결 절차
- **C2**: 네이버 서치어드바이저 등록 + 사이트맵 제출
- **C3**: 구글 Search Console 등록
- **C4**: 기본 SEO 체크 (현재 index.html의 meta 태그 검토)
- 형태: `GUIDE/도메인_SEO_가이드.md`
